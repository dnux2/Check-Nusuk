CheckNusuk — Smart Hajj & Pilgrim Management Platform

![WhatsApp Image 2026-03-11 at 7 45 33 AM](https://github.com/user-attachments/assets/34b00a1d-f086-446e-981f-6fc3353a7f9e)

A full-stack web application built for managing Hajj pilgrims in real time.
Features include live location tracking on an interactive Leaflet.js map,
AI-powered route analysis (Valhalla/OSRM + GPT-4o), TensorFlow.js camera 
detection, emergency response system, AI translation with TTS, real-time 
WebSocket chat between supervisors and pilgrims, and a bilingual (AR/EN) 
RTL-first interface with an emerald green Nusuk design.

Two portals: Supervisor Dashboard (/login) · Pilgrim App (/login?tab=pilgrim)
